# Simple_library
## Консольное приложение для управления библиотекой книг

Приложение было создано и протестировано на Python 3.12 в операционной системе Windows 11 Домашняя версии 23H2.

Чтобы запустить приложение, надо перейдите в корневой каталог приложения и ввести команду:

```python app.py```

После запуска приложения в консоли появиться главное меню, состоящее из пяти пунктов:

1. Добавление книги.
2. Удаление книги.
3. Поиск книг.
4. Отображение всех книг.
5. Изменение статуса книги.

Выйти из приложения можно введя в консоли 'q' или 'quit'. 
Для выбора действия надо ввести в консоли соответствующий пункт меню.

### Добавление книги
Для добавления книги в консоли надо выбрать пункт меню 1. Далее, для добавления книги в библиотеку, 
будет по очереди предложено ввести наименование книги, автора книги и год издания книги. В случае 
некорректного ввода каких-либо из данных, будет выведено предупреждение об ошибке ввода, и предложено ещё раз 
попробовать ввести данные или отменить ввод, введя в консоли 'c' или 'cancel'. В случае отмены ввода произойдёт
возврат в главное меню, и книга создана не будет. 

Для корректного ввода, заголовок книги должен быть длинной от 3 до 50 символов; автор книги должен быть длинной 
от 2 до 25 символов; год издания должен быть целым числом, значение которого не превышает значение текущего года.

При корректно введённых запрошенных данных в консоли отобразиться сообщение о добавлении новой книге, и будет 
указан назначенный её уникальный числовой идентификатор. Статус книги при этом будет установлен как 'available' (доступная).

### Удаление книги
Для удаления книги в консоли надо выбрать пункт меню 2. Далее будет предложено ввести уникальный числовой 
идентификатор книги, которую требуется удалить. В случае некорректного ввода идентификатора, будет выведено 
предупреждение об ошибке ввода, и предложено ещё раз попробовать ввести идентификатор или отменить ввод, 
введя в консоли 'c' или 'cancel'.

Для корректного ввода, идентификатор должен быть целым положительным числом.

При попытке удалить книгу с несуществующим идентификатором, на консоли появиться сообщение, что книги 
с указанным идентификатором не найдено, и произойдёт переход к главному меню.

В случае ввода идентификатора книги, которая присутствует в библиотеке, на консоли появиться запрос на подтверждение
удаление книги. Так же будет отображена подробная информация о книге, которая будет удалена при подтверждении 
её удаления. В качестве подтверждения удаления надо вести в консоли 'y' или 'yes', или для отказа от удаления книги 
в консоли надо ввести 'n' или 'no'. Если ввести неправильное подтверждение, то будет отображено предупреждение 
о некорректном вводе подтверждения, и предложено ещё раз попробовать ввести идентификатор или отменить ввод, 
введя в консоли 'c' или 'cancel'.

Если выбрать отказ от удаления книги, то на экране появиться сообщение, что книга, с указанным Вами ранее 
идентификатором, удалена не будет, и произойдёт возврат в главное меню.

Если же выбрать подтверждение удаления книги, то на экране появиться сообщение, что книга с указанным ранее номером 
была удалена. Далее произойдет возврат к главному меню.

### Поиск книг
Для поиска книг в консоли надо выбрать пункт меню 3. После этого появиться следующее подменю с предложением выбрать критерий 
для поиска книг:

1. Наименование.
2. Автор.
3. Год издания.

При попытке выбора несуществующего меню, будет выведено предупреждение об ошибке ввода, и предложено ещё раз попробовать 
выбрать пункт меню критерия поиска или отменить ввод, введя в консоли 'c' или 'cancel'.

Для поиска книг по наименованию надо выбрать пункт меню 1. После этого будет предложено ввести наименование книги, 
которую требуется найти, или часть наименования, чтобы найти все книги, в наименование которых входит искомая строка.
Поиск книг по наименованию ведётся без учёта регистра.

Для поиска книг по автору надо выбрать пункт меню 2. После этого будет предложено ввести автора, книги которого
требуется найти. Для корректного поиска автор вводится полностью, как указано в книге и с учётом регистра.

Для поиска книг по году издания надо выбрать пункт меню 3. После этого будет предложено ввести год издания, книги 
которые требуется найти. В случае некорректного ввода года издания, будет выведено предупреждение об ошибке ввода, 
и предложено ещё раз попробовать ввести год издания или отменить ввод, введя в консоли 'c' или 'cancel'. В случае 
отмены ввода произойдёт возврат в главное меню.

Для корректного ввода год издания должен быть целым числом, значение которого не превышает значение текущего года.

При удачном поиске по выбранному критерию, в консоли будет отображено количество найденных книг, а также список всех
книг в виде подробной информации о каждой найденной книге.

### Отображение всех книг
Для отображения всех книг находящихся в библиотеке в консоли надо выбрать пункт меню 4. После этого появиться 
уведомление об общем количестве книг, находящихся в библиотеке, а также список всех книг, в виде подробной информации 
о каждой книге.

### Изменение статуса книги
В библиотеке книга имеет два статуса <ins>available</ins> (доступная) и <ins>given out</ins> (выданная). Для изменения статуса книги 
в консоли надо выбрать пункт меню 3. Далее будет предложено ввести идентификатор книги, статус которой надо изменить.
В случае некорректного ввода идентификатора, будет выведено предупреждение об ошибке ввода, и предложено ещё раз 
попробовать ввести идентификатор или отменить ввод, введя в консоли 'c' или 'cancel'. В случае отмены ввода произойдёт 
возврат в главное меню.

После ввода идентификатора книги, статус которой требуется изменить, будет запрошен ввод нового статуса книги. 
В случае некорректного ввода статуса книги, будет выведено предупреждение об ошибке ввода, и предложено ещё раз 
попробовать ввести статус книги или отменить ввод, введя в консоли 'c' или 'cancel'. В случае отмены ввода произойдёт 
возврат в главное меню.

Для корректного ввода, для изменения статуса на <ins>доступная</ins>, надо ввести 'a' или 'available'. Для корректного 
ввода, для изменения статуса на <ins>выданная</ins>, надо ввести 'g' или 'given_out'.

После корректного ввода статуса книги, если был указан идентификатор, которого в библиотеке нет, то будет выдано 
сообщение, что книга с указанным идентификатором отсутствует. А если был указан существующий идентификатор, то
появится уведомление о том, что книга с указанным идентификатором теперь имеет выбранный Вами статус.
