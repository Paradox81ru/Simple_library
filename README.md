# Simple_library
## Консольное приложение для управления библиотекой книг

Приложение было создано и протестировано на Python 3.12 в операционной системе Windows 11 Домашняя версии 23H2.

Чтобы запустить приложение, надо перейдите в корневой каталог приложения и ввести команду:

```python app.py```

После запуска приложения в консоли появиться меню, состоящее из пяти пунктов:

1. Добавление книги.
2. Удаление книги.
3. Поиск книг.
4. Отображение всех книг.
5. Изменение статуса книги.

Выйти из приложения можно введя в консоли 'q' или 'quit'. 
Для выбора действия надо ввести в консоли соответствующий пункт меню.
